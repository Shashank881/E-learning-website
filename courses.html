C Programing Projects

C is a general-purpose, procedural programming language developed by Dennis M. Ritchie at Bell Laboratories in the early 1970s. It was primarily designed for system programming, particularly for writing the UNIX operating system.

C++ Programing Projects

C++ originated in 1979 when Bjarne Stroustrup, a Danish computer scientist at AT&T Bell Laboratories, began developing "C with Classes." His goal was to enhance the C programming language with object-oriented programming (OOP) features like classes, inheritance, and polymorphism.

Javascript

JavaScript started as a prototype scripting language written by Brendan Eich in 1995 for the Netscape Navigator browser under the working name of “Mocha”. After being renamed to “JavaScript”, it was standardized by ECMA International as ECMA-262, or ECMAScript in 1997.

HTML

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript)
